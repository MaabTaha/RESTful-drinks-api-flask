# RESTful Drinks API
This project was built while following Caleb Curry's epic YouTube tutorial: [REST API Crash Course - Introduction + Full Python API Tutorial](https://youtu.be/qbLc5a9jdXo?si=3OUI-Nyer2vdQ5E3)

The majority of the tutorial focuses on building a REST API using Flask and SQLAlchemy. I highly recommend it for anyone new to the world of APIs.


## What This Project Includes

### Backend API

* Flask REST API
* SQLite database
* SQLAlchemy ORM for interacting with the database

### API Consumption

* Python script that consumes the Stack Overflow API
* Basic filtering of API responses

### Testing

* Postman for testing endpoints and sending requests

---

## Supported Operations

| Method | Endpoint       | Description            |
| ------ | -------------- | ---------------------- |
| GET    | `/drinks`      | Retrieve all drinks    |
| GET    | `/drinks/<id>` | Retrieve a drink by ID |
| POST   | `/drinks`      | Create a new drink     |
| DELETE | `/drinks/<id>` | Delete a drink         |
